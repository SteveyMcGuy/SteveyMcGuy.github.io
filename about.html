About SteveyMcGuy

Who am I?

SteveyMcGuy (aka Joshua Wedgwood, or Bobscorn) is a small, passionate indie game developer, and programmer.

What do I do?

Currently making games.
Small indie games for now, but with a passion.
I don't like settling for copy-paste games, with exorbitant prices or smothered by ads.
We enjoy making special hand crafted games that are enjoyable to play.

What have I done?

I have spent the last few years doing a variety of programming.
In chronological order, this includes:

  • My own C++ Game Engine
  • A light-hearted Trump co-op game - Attack on Trumpocracy
  • A PNG/JPEG -> SVG image processor - Vectorizer
  • A 3D physics stacking Mobile Game - Piece Stacker 3D
  • A Peer-to-peer UDP File Transfer Tool - PTOP
Previously I've spent 2.5 years making my own C++ game engine using SDL2, OpenGL, OpenAL and a few other libraries trying to do myself what all these triple AAA games were doing.
I was able to make a prototype game with this custom game engine, but not a completed product.

I then tried out Unity and made a few games, namely Super Farm (my first game, likely to never see the light of day), Attack On Trumpocracy, and Piece Stacker 3D (aka Stacker).

Otherwise I've been honing my general programming skills with various projects like PTOP - a peer to peer file transfer program, and Vectorizer - a PNG/JPG -> SVG image processor
